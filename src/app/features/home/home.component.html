How To Play

1

Choose Your Level

Select from Easy, Medium, or Hard difficulty levels. Each difficulty has five progressively challenging mazes.

2

Navigate The Maze

Use the arrow keys or on-screen controls to move through the maze. Find the path from the start (green) to the end (orange).

3

Utilize BFS

Stuck? Use the BFS algorithm to find the shortest path (limited to 3 uses per level).
